Why Aura?

Effortless Dependency Management

Aura simplifies the process of integrating external libraries into your C++ project. Using the power of vcpkg, Aura can automatically install, configure, and manage libraries for you, so you no longer have to worry about version mismatches or missing dependencies. Whether you're working with popular libraries or niche ones, Aura ensures a smooth integration process.

Cross-Platform Compatibility

With vcpkg, Aura guarantees that your libraries are compatible across all major platforms—Windows, Linux, and macOS—making it easier to work in any environment without the hassle of manual configurations.

Quick Environment Setup

Set up your entire C/C++ development environment effortlessly with just a single command. Aura streamlines your workflow by quickly installing and configuring the necessary tools, libraries, and settings to help you focus on writing code, not managing environments.

Integrated Unit Testing

Running unit tests for your C and C++ projects has never been easier. Aura provides seamless integration with popular testing frameworks, allowing you to quickly set up and execute tests, ensuring that your code is always reliable and ready for deployment.

Automated Project Setup

Generate a new C/C++ project structure with ease. Aura creates a standardized and well-organized directory layout, allowing you to focus on building your project rather than setting up folders, files, and configurations. This automated setup ensures consistency across all your projects.

Best Practices

Even your simplest Hello-World application will follow best practices when using Aura. It provides scalable, modular project structures that grow with your needs. Install external libraries without even touching your CMakeLists.txt file. Aura ensures that your project stays clean, maintainable, and production-ready.

Seamless CMake Integration

Aura integrates smoothly with CMake, eliminating the need for manual configuration adjustments. When you add new dependencies through vcpkg, Aura automatically updates your CMake setup, saving you time and effort in maintaining your build system.

Version Control Integration

Aura helps you manage version control for your projects easily. With built-in support for Git, it can automatically track changes to your project files, dependencies, and CMake configurations. This ensures that your development process stays organized and efficient, from initial setup to final deployment.

Continuous Integration Support

Integrate Aura with your continuous integration (CI) pipeline effortlessly. Aura provides easy-to-use tools for automating build processes, testing, and deployment, making it perfect for teams that need reliable and repeatable workflows.
